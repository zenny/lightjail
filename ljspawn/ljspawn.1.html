NAME

ljspawn - spawn a process in a lightweight jail environment

SYNOPSIS

ronn [-i IP address] [-n name] -d directory -p process

DESCRIPTION

ljspawn runs the process in a FreeBSD jail under directory with an optional IP address and name.

It was made for running apps in a lightweight, ephemeral way, without any config files.

OPTIONS

-i IP address: The IP address that the jail will have. It will also be added as an alias to the lo0 network interface.

-n name: The hostname and jail name that the jail will have.

-d directory: The directory that contains the jail environment. At least, it must have a FreeBSD base system (world), installed by make -C/usr/src installworld DESTDIR=<directory>. But a lot of applications might require a devfs. The recommended way to prepare a directory for ljspawn is:
1. mount a world using nullfs in read-only mode
2. mount devfs with a limited set of devices
3. mount the application using unionfs in read-only mode
4. mount a temporary directory using unionfs in read-write mode

-p process: The command that will be run in the jail environment. Technically optional, but will be echo 'Hello world' by default, which is not very useful.
